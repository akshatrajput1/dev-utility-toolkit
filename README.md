# Dev Utility Toolkit 🚀

A beginner-friendly JavaScript project to practice ES6 modules using default and named exports.

## 🔧 Features
- String Formatter (default export)
- Math Utilities (named exports)
- Array Operations (mixed exports)
- Interactive UI

## 🧠 Concepts Covered
- ES6 Modules
- Import / Export
- Default vs Named Exports
- Code modularization

## 📂 Project Structure
dev-utility-toolkit/
│── index.html
│── main.js
│── mathUtils.js
│── stringUtils.js
│── arrayUtils.js
│── style.css

## ▶️ How to Run
Use Live Server in VS Code or any local server.
